=========
Catalogue
=========

This is an essential IZI app which exposes functionality to manage your
product catalogue.
:class:`izi.apps.catalogue.abstract_models.AbstractProduct`
is its main model.
The catalogue app also includes views specific to viewing a list or
individual products.
